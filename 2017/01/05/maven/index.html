Maven

发表于 2017-01-05   |   分类于 技能点总结

​ Apache Maven是一个软件项目管理和综合工具。基于项目对象模型（POM）的概念，Maven可以从一个中心资料片管理项目构建，报告和文件。本文将介绍Maven的功能，以及如何使用。

Maven的功能

1.项目目录管理

2.通过配置文件pom.xml，可以将需要的jar包在网站上下载到本地仓库

3.避免相同代码的重复

3.Maven类型

Maven有三种类型：pom,jar,web

pom:文件夹,(只能放两种类型的文件,分别是jar和war)

jar:java项目,将来要打包成jar包

web:web项目,将来要打包成war包

配置编译环境

1.pom.xml

2.依赖管理器

​ 没有使用依赖管理器时，子项目会继承父项目pom.xml中声明的全部jar包，但是存在一些jar包多余，未使用到的情况，所以使用依赖管理器，父项目只管声明，子项目只管使用。

（1）.dependencyManagement

父项目只管声明,声明的时候带上版本号;是由项目经理或者项目的负责人来维护版本号

（2）.子项目只管使用,使用的时候不要带版本号,自动的读取父项目中的版本号;只使用你用的到的jar包

3.项目依赖

web项目可以依赖java项目，也可以java项目依赖java项目

4.web项目运行

直接借助于Eclipse自带的tomcat运行
