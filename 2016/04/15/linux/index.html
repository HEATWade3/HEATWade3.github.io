Linux命令总结

发表于 2016-04-15   |   分类于 技能学习

​ 本文总结了一些在Linux系统使用到的shell命令……

目录/文件查询： （绝对路径或相对路径，不加表示当前目录， h表示文件大小方便看懂，会列出目录下面的文件或者文件夹）

ll -h 路径    ll -h /
ls            ls -lh

查看当前目录(当前在哪 绝对路径)：
pwd
创建目录：(目录名：在当前目录下创建；绝对路径：可以在其他目录下创建)
mkdir 目录
修改目录：（原始目录要存在，目标目录不存在，移动原始目录并改名；目标目录存在，剪切原始目录到目标目录下面）
mv 原始目录 目标目录
切换目录：
cd 路径（绝对或相对）
删除目录：
rm -rf 路径（r表示递归，f表示强制 ，绝对或相对，文件）

退出：先按esc键 然后输入 :wq 保存并退出 :q!不保存退出
../上一层目录 .当前目录
先看历史：history
help+man: –help ;

文件创建：
touch 文件所在的路径（绝对或相对）
修改文件：（1.目标文件 将源文件重命名为目标文件；2.目标目录 将源文件移动至目标目录）
mv 源文件 目标
文件内容修改：
如果文件名存在，修改文件名；如果文件名不存在，表示创建一个文件
查看文件
vi 文件路径名 ；less 文件路径名 ;more 文件路径名 ;cat 文件路径名
：查找模式
set nu (显示行号)
/ 关键字 按下n键查找下一个 按下N查找上一个
ctrl+f 向下翻页 ctrl+b 向上翻页
复制：（如果是一个目录，要加上 -r递归）
cp 源目录（文件） 目标目录（文件）

创建用户：
useradd 用户名
修改密码： passwd 用户名
删除用户： userdel 用户名

查询文件
find / -name my.cnf

解压
unzip
